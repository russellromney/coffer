Your secrets.

A self-hosted secrets manager CLI. Store encrypted secrets locally, inject them into your apps at runtime. Zero monthly fees.

Get Started GitHub

Go Install

go install github.com/russellromney/coffer@latest

From Source

git clone https://github.com/russellromney/coffer.git
cd coffer && make build
sudo mv coffer /usr/local/bin/

Download Binary

# macOS, Linux, Windows binaries available
github.com/russellromney/coffer/releases

Quick Start

$ coffer init
Enter master password: ********
Vault created at ~/.coffer/vault.db

$ coffer project create myapp && coffer project use myapp
Project 'myapp' created and selected

$ coffer env create dev
Environment 'dev' created

$ coffer set DATABASE_URL "postgres://localhost/myapp" --env dev
Secret 'DATABASE_URL' saved

$ coffer run --env dev -- npm start
Injecting 1 secret into environment...
